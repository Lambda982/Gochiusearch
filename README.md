# ごちうさ全動画検索エンジン Gochiusearch
ご注文はうさぎですか？1期・2期の全画像を対象とした検索エンジンです。
アプリケーションにキャプ画像をブラウザやフォルダからDrag&Dropすると、
何羽のどのあたりかを返します。
また、自動的にニコニコ動画の該当部分に移動することもできます。
